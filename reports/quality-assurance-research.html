What does quality assurance mean for web site development?
Quality Assurance (QA) ensures that end-users get a functional user interface and the best user experience when using a web app or website. It examines a website or web application to try and uncover any flaws that might have been overlooked during design and development. Quality Assurance usually involves:
  • Requirement testing
  • Design evaluation
  • Functionality testing
Reference (F5Studio)
Describe the two key principles of Quality Assurance: "Fit for Purpose" and "Right First Time"
  • Fit for Purpose

    • The product or service meets its intended purpose

  • Right First Time

    • Any mistakes are eliminated and/or immediately dealt with

Reference (ProjectManager.com)
How do web standard relate to web quality?
Web standards are rules and guidelines that can be used to achieve web quality
Describe what Web Accessibility means.
Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them
Reference (w3 Accessibility Intro)
What steps you can take to make your pages more accessible?
  • Provide text alternatives for non-text content

    • Text alternatives convey the purpose of an image or function to provide an equivalent user experience

  • Ensure that content can be presented in different ways

    • This allows content to be correctly read aloud, enlarged, or adapted to meet the needs and preferences of different people

  • Ensure functionality is available from a keyboard

    • Many people do not use the mouse and rely on the keyboard to interact with the web

  • Ensure text is readable and understandable

    • Content authors need to ensure that text content is readable and understandable to the broadest audience possible, including when it is read aloud by text-to-speech.

  • Ensure content is compatible with current and future user tools

    • Meeting this requirement helps maximize compatibility with current and future user agents, including assistive technologies.

Reference (w3 Accessibility Principles)
