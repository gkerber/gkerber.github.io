How does a search engine work?
A search engine uses a three step process:
  1. Web Crawling

    2. The search engine determines what has been published on the web

  2. Indexing

    3. A repository of web pages returned from web crawling is stored in a data center. This data store is organized and used to provide the search results that are returned.

  3. Algorithm

    4. An algorithm is a complex equation which calculates a value for any given site in relation to a search term

Why do different search-engines give different ranking to websites for the same search phrase?
How the web is crawled, how data is indexed, and the algorithm used to calculate a value for a given site is unique to each search engine so results will vary in accordance with the different way of collecting, organizing, and ranking search results

What is a <meta> tag?
Metadata is data about data. The <meta> tag provides metadata about the HTML document (i.e. page description, keywords, author of the document, last modified).
Reference
What is the purpose for Keyword and Description meta tags?
The meta Keywords is an HTML attribute that helps tell search engines what the topic of a page is
Reference
The meta Description is an HTML attribute that provides a brief summary of a web page

Why might you include meta tags in your pages?
Meta tags might be included to provide data about a page to search engines and website visitors.
