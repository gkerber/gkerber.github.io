Short Answers To Research Questions

What is an internet domain name?

A domain name is a unique set of characters that identifies a specific website.

When you enter a domain name into a web browser, the browser accesses a domain name server (DNS) to find the location of the corresponding website on the internet, so that it can retrieve the website and display it to you

Reference

Why do domain names have to be unique?

A domain name must be unique because it associates IP address(es) on one or more DNS Server(s) hosting a single web site. If it was not unique, the server(s) would not know which web site to return when the domain name was typed in the browser.

What does it mean to "register" a domain name?

Registering a domain name means mapping the domain name to IP address(es) and other information associated with the domain in the DNS (Domain Name System)

Reference

If you wanted a domain name that is already taken, how could you get it?

  1. Buy it from the current owner
  2. Wait for it to expire and register it when it becomes available
  3. Register the domain name with a different extension if it is available (i.e. .net instead of .com)
