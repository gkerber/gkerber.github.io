Project plan

Title

I Have A Havanese

Goal

The goal of this website is to share information about Havanese so that people can learn about the particular breed of dog.

Target Audience

People who have a Havanese dog or are interested in owning a Havanese dog

Main Features
  • History of the breed
  • Breed characteristics
  • Typical health issues
  • Photos
Personality
  • Energetic
  • Happy
  • Playful
Limiting Factors

No experience with Photoshop (Required?)

Styles guide

Background

Background is cream

{background-color: #D8CAA8}

Default font

Text is dark green, 12 pixels

p { font-size: 12px; color: #284907;}

Header

The header section is 200px tall, positioned along the top of each page with the logo centered in the frame. There is clip art of Havanese dogs framing the logo. The background is brown. There is a border along the bottom.

header {
height: 200px;
background-color: #382513;
border-bottom: .3rem solid #5C832F;
text-align: center;
margin: 2rem;
padding: 1rem;
}

Navigation

The main navigation is positioned just below the header with a border color of yellow.

nav { border: 4px solid #FFFFCC; }

Images

Images will be over gray background with a bright green border around the container

figure { background-color: #363942; padding: 1rem; border: 4px solid #5C832F; }

Links

Links will be blue. Visited links will be purple. On hover will be red. Active links will be red with a black background.

a:link { color: #0000FF; }

a:focus { outline: thin dotted; } * Per Lynda video instructions

a:visited { color: #800080; }

a:hover { color: darkred; background-color: black; text-decoration: none; outline: 0; }

a:active { color: black; background-color: darkred; text-decoration: none; outline: 0; }
